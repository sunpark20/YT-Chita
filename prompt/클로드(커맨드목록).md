# 커스텀 슬래시 커맨드 목록

`.claude/commands/` 폴더에 마크다운 파일로 정의됨.

---

## /project:c-github-build

- **파일**: `.claude/commands/c-github-build.md`
- **요약**: GitHub Actions에서 인텔맥/실리콘맥/Windows 10,11(64bit) 빌드 시 주의사항 (러너, entitlements, 태그 관리, 버전 올리기)
- **실행**: 대화창에 `/project:c-github-build` 입력
- **용도**: 새 버전 빌드 & 릴리즈할 때

---

## /project:c-log-debug

- **파일**: `.claude/commands/c-log-debug.md`
- **요약**: 오늘 날짜의 yt-dlp 로그 파일(`~/Library/Logs/YouTubeDownloader/app_YYYYMMDD.log`)을 읽고 ERROR/WARNING 분석
- **실행**: 대화창에 `/project:c-log-debug` 입력
- **용도**: 다운로드 후 에러 원인 파악할 때
