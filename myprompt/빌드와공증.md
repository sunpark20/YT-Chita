# 빌드 + 코드 서명 + 공증 (Apple Silicon)

## 1. 빌드 + 코드 서명 (한방)
```bash
bash build.sh
```

## 2. 공증 키체인 프로필 저장 (최초 1회)
```bash
xcrun notarytool store-credentials "notarytool" --apple-id "sun.park20@gmail.com" --team-id "GA2LMK5XL2" --password "앱전용비밀번호"
```

## 3. 공증 제출
```bash
ditto -c -k --keepParent dist/YouTubeDownloader.app dist/YouTubeDownloader.zip
xcrun notarytool submit dist/YouTubeDownloader.zip --keychain-profile "notarytool" --wait
```

## 4. 스테이플
```bash
xcrun stapler staple dist/YouTubeDownloader.app
```

## 5. 배포용 DMG 만들기 (선택)
```bash
hdiutil create -volname "YouTubeDownloader" -srcfolder dist/YouTubeDownloader.app -ov -format UDZO dist/YouTubeDownloader.dmg
```
