오늘 날짜의 yt-dlp 로그 파일을 읽고 에러/경고를 분석해줘.

사용자 추가 메시지: $ARGUMENTS

---

## 로그 파일 위치

`~/Library/Logs/YouTubeDownloader/app_YYYYMMDD.log` (YYYYMMDD는 오늘 날짜)

## 수행할 작업

1. 오늘 날짜의 로그 파일을 Read 도구로 끝부분 200줄 읽기
2. `ERROR`, `WARNING`, `yt-dlp` 관련 메시지를 찾아서 정리
3. 문제가 있으면 원인과 해결 방법 제시
4. 문제가 없으면 "이상 없음"으로 간단히 보고
5. 사용자가 추가 메시지를 남겼으면 그 맥락에 맞춰 분석
